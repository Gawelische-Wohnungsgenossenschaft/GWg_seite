第一章 総則

第一条（名称）

本組合は「ガウェラ住宅協同組合（ガウェラジュウタクキョウドウクミアイ）」と称する。

2. ドイツ語名は „Gawelische Wohnungsgenossenschaft“ とし、略称を GWg とする。

第二条（目的）

本組合は、営利を目的とせず組合員の相互扶助を基本とする、民主的な協同組合である。管理運営を通じて組合員の住環境を向上させ、ガウェラ特別州における協同組合精神の普及と美しい都市景観の維持を目的とする。

第三条（事業内容）

本組合は、協同組合住宅の管理運営等を行う。

第四条（所在地）

本組合の主たる事務所を、ガウェラ特別州のロースハウス / DG に置く。

第二章 組合員

第五条（加入資格）

ガウェラ特別州の住人であり、以下の要件を満たす者は組合員となることができる。

  • 一． 民主主義に基づいた協同組合精神を尊重すること。
  • 二． 組合総会の決定事項を遵守すること。

第六条（加入手続き）

加入希望者は代表理事に申し出るものとし、組合の趣旨への賛同を表明し、代表理事の承認を得ることで加入を認める。

第三章 運営と組織

第七条（組合総会と決議プロセス）

総会は組合の最高意思決定機関であり、以下のプロセスに従って運営される。

2. 総会は、全組合員の3分の2以上の出席をもって成立する。

3. 議案の決定は、以下の段階を追う。

  • 一． すべての議案は、原則として全会一致をもって決定する。
  • 二． 議案に対して相当期間の議論を経ても合意に至らない場合、代表理事は48時間以上の熟慮期間を設けることができる。
  • 三． 期限に達した時点で全会一致に至っていなかった場合、代表理事は出席者の3/4以上の賛成による決議に切り替えるか、議案を廃案にできる。

4. 緊急を要する維持管理については、代表理事の判断で暫定的な措置を行い、事後に総会で承認を得ることができる。

第八条（代表理事）

代表理事の任期は実時間で3ヶ月とする。

2. 候補者名簿には、組合加入から一週間を経過した全ての組合員が登録・消去できる。創設者は、永続的に候補者名簿に登録され、組合の理念維持に努める。

3. 代表理事は、以下の方法をもって選出する。

  • 一． 代表理事の任期満了、あるいは空席に伴い代表理事の選出が開始される。
  • 二． 代表理事は候補者名簿から組合員名簿のMCIDに依るアルファベット・数字昇順に基づき輪番で選出される。ただし、やむを得ない事由により就任が困難な場合は、次順位者への繰り下げを認める。
  • 三． 代表理事が不在かつ次順位者もいない場合、総会にて臨時代表理事を選出する。

4. 代表理事が組合の理念に反する行為を行った、または運営に著しい支障をきたすと判断される場合、全組合員の3分の2以上の賛成をもって、任期中であっても当該代表理事を解任することができる。

5. 代表理事が7日以上連絡なく不在の場合、組合員は総会において代表理事選出の動議を提出することができる。出席者の半数以上の賛成が得られた場合、新たな代表理事を所定の方法に則り選出する。

第四章 住宅の管理と利用

第九条（住宅の新設と認可）

組合が管理する新たな協同組合住宅を建設しようとする場合は、事前にその設計案および建設予定地について総会に提出しなければならない。

2. 組合住宅の新設および既存建物の組合住宅への編入は、第六条に定める総会の決議をもって認可される。

第十条（入居および割当）

組合員は、組合が管理する住宅のうち、空き住戸について入居を申請することができる。

2. 代表理事は、前項の申請に対し、速やかに当該住戸の割り当てを行うものとする。

3. 住戸の割当は、原則として申請順とするが、特別な事情がある場合は総会の決議により調整を行うことができる。

第十一条（専有部分の利用と制限）

組合員は、割り当てられた住戸の内装を自由に変更できる。ただし、建物の構造や隣接部分、外部環境に影響を与えない範囲に限る。

2. 住戸の利用は、社会通念上の常識に沿った範囲内であれば、住居以外の用途（工房、店舗等）としても利用できる。

3. 組合員は、割り当てられた住戸を第三者に貸し出し、または居住させることはできない。

第十二条（代執行）

組合員が不適切な改築を行い、代表理事の是正勧告に従わない場合、組合は当該箇所の代執行を行うことができる。この際、原状回復に要した資材等の費用は当該組合員の負担とする。

第五章 会計と負担

第十三条（資金形態）

本組合は家賃および月額組合費を徴収しない。

2. 本組合の運営維持費は、原則として組合員および外部からの寄付によって賄われる。

3. 入居時に、安価に設定された「加入金」を支払うものとする。

第十四条（労働および拠出）

総会で決定された共有スペースの整備、建築プロジェクト、破損等の修繕、または資金の拠出について、組合員は「努力義務」としてこれに協力するものとする。

第六章 退去、脱退および譲渡

第十五条（任意脱退）

組合員は、代表理事に申し出ることで、いつでも自由に組合を脱退できる。

2. 住戸の利用権を他者に譲渡する場合は、既存の組合員間においてのみ可能とする。新規加入者への譲渡を希望する場合は、当該人物が第四条の手続きを経て加入を認められることを条件とする。

3. 代表理事への連絡なく、コムザール連邦共和国に所属しなくなった組合員は、脱退したものとみなし、総会の承認を経て住戸の利用権を回収できるものとする。

第十六条（強制退去）

組合員が本規約に著しく違反し、改善の見込みがない場合、組合総会の決議に基づき強制退去を執行することができる。

2. この際、建物構造体および外装に使用された資材の所有権は組合に帰属するものとし、原則として搬出を禁止する。内装品やチェスト内の私物は範囲外とし、強制退去執行後実時間で7日以内に持ち出さなければならない。

3. 前項の期限を過ぎた残置物については、組合がその所有権を取得し、処分または再利用できるものとする。

第七章 雑則

第十七条（解散時の措置）

本組合が解散する場合、組合が保有する全ての土地、建物および資産はガウェラ特別州へ帰属させるものとする。

第十八条（細則への委任）

本規約に定めのない事項、および運営に必要な具体的規則については、代表理事あるいは総会の決議を経て別途「管理細則」を定めることができる。

第十九条（規約の改定）

本規約は、総会の全会一致により変更できる。

第二十条（国内法への委任）

本規約に定めのない事項については、ガウェラ特別州条例および連邦法を優先して適用する。

附則

（施行期日）

本規約は、2026年2月25日より施行する。

（初代代表理事・創設者の指名）

本組合の設立時における代表理事は、創設者である I_korobka （カロッカ）とする。

（法人法上の責任者）

本組合の法人法上の事業責任者は、代表理事とは独立して創設者とする。法人法上の事業責任者は総会での議決により変更できる。
